# Code of Conduct

Software Engineering Project – Booze Buddies

## 1. Purpose

The purpose of this Code of Conduct is to ensure a welcoming, inclusive, and respectful environment for everyone participating in the Booze Buddies project.

## 2. Expected Behavior

All participants are expected to:

- Be respectful and considerate of different perspectives
- Use welcoming and inclusive language
- Provide constructive feedback during discussions and reviews
- Collaborate openly and act with integrity
- Respect others’ privacy and personal boundaries

## 3. Unacceptable Behavior

The following behaviors are considered unacceptable:

- Harassment, bullying, or discrimination of any kind
- Use of sexualized or offensive language or imagery
- Personal attacks or insults
- Disruptive or unprofessional conduct in project spaces (including online discussions and meetings)

## 4. Scope

This Code of Conduct applies to all project spaces, including but not limited to:

- Online discussions, code reviews, commits, and issue tracking
- Public representations of the project (e.g., social media, conferences, or meetups)
- Any situation where an individual is acting as a representative of the Booze Buddies project

## 5. Reporting and Enforcement Guidelines

If you experience or witness behavior that violates this Code of Conduct, you are encouraged to report it to a project maintainer via email. Reports should include:

- A description of the incident
- The date and time it occurred
- Any supporting documentation (screenshots, logs, etc.)

Project maintainers will:

- Handle all reports respectfully and confidentially
- Review the report with at least two maintainers
- Take appropriate action based on the severity and nature of the violation

## 6. Enforcement

Maintainers may take actions including, but not limited to:

- Private warnings or reminders of expected behavior
- Temporary suspension from collaboration or project spaces
- Permanent removal from the project in severe cases
